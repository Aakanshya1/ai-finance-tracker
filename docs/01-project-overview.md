# AI Expense Tracker- project Overview

This project is a full-stack personal tracking application designed to help users record, analyze and understand their daily financial activities.


This application allows users to securely log income, expenses, categorize incoming , categorize transactions, track savings, and view monthly financial summaries. An AI layer is integrated  to generate human-readable insights, spending summaries, and personalized saving suggestions.

The primary goal of this project is to demonstrate clean full-stack architecture, real-world business logic, and meaningful AI integration rather than simple CRUD operations.



# Tech Stack

Frontend: React + Tailwind CSS
Backend : Node.js + Express
Database: MongoDB
Authentication: JWT
AI: OpenAI API ( express insights & financial summaries)
